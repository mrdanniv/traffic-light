# Traffic Light Interactive

Веб-приложение для демонстрации навыков работы с DOM-деревом, событиями и адаптивной версткой.

## Функционал
- Управление фоном страницы через интерактивную панель.
- Динамическая генерация случайных градиентов для элементов второго ряда.
- Кнопка сброса состояния к исходным настройкам.
- Поддержка Pointer Events для корректной работы на сенсорных экранах.

## Стек технологий
- HTML
- CSS (Flexbox, Grid, адаптивные медиа-запросы)
- JavaScript (ES6+, работа с событиями ввода)

## Инструкция по установке
1. Склонируйте репозиторий:
   git clone https://github.com/mrdanniv/traffic-light.git
2. Откройте index.html в браузере.
